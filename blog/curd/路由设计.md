# 路由设计

| 请求方法 | 请求路径 | get 参数 | post 参数 |                   备注 |
| -------- | -------- | :------: | --------- | ---------------------: |
| GET      | /        |          |           |               渲染首页 |
| GET      | /add     |          |           |       渲染学生添加页面 |
| POST     | /add     |          | 学生数据  |           提交学生数据 |
| GET      | /edit    |    id    |           |           渲染编辑页面 |
| GET      | /remove  |    id    |           | 删除当前学生并且重定向 |
